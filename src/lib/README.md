# Library (`lib/`)

Utility functions and shared logic that can be used across the application.

## Purpose
- Helper functions
- Data formatting/transformation
- Validation functions
- API utilities
- Common algorithms

## Example Structure
```
lib/
├── api/
│   └── client.ts
├── formatting/
│   └── date.ts
└── validation/
    └── email.ts
```

Keep functions small, testable, and focused on a single responsibility.
