# Styles (`styles/`)

Global and shared stylesheets.

## Purpose
- Global CSS/SCSS files
- Theme definitions
- Utility styles
- Shared animations

## Structure
```
styles/
├── globals.css
├── variables.css  (CSS custom properties)
├── animations.css (Keyframes and animations)
└── utilities.css  (Utility classes)
```

Note: Component-specific styles should remain with their components using CSS Modules or inline styles.
